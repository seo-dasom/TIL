# 목록 관련 태그

```html
<ul></ul>
    순서가 필요하지 않은 목록(unordered list) 만들 때 사용
    리스트 앞에 특정 모양 출력(기본 값∵), 모양은 CSS 이용해 수정 가능

    [예시]
      <ul>
        <li>내용1</li>
        <li>내용2</li>
        <li>내용3</li>
      </ul>
```

```html
<ol></ol>
    순서가 있는 목록(ordered list) 만들 때 사용
    속성으로 문자(A, B, C.../a, b, c...), 숫자(1, 2, 3...),
    로마자(Ⅰ, Ⅱ, Ⅲ.../ⅰ, ⅱ, ⅲ...) 설정
    CSS로 순서 설정 가능

    [예시]
      <ol type="설명문자" start="시작순서">
        <li>내용1</li>
        <li>내용2</li>
        <li>내용3</li>
      </ol>
```
|속성|내용|
|---|---|
|type|순서형식을 정하는것|
|Start|시작 순서|

```html
<dl></dl>
    용어나 문장에 대한 정의(definition) 리스트로 자동으로 들여쓰기가 됨

    [예시]
      <dl>
        <dt>내용1</dt>
        <dd>내용2</dd>
        <dd>내용3</dd>
      </dl>
```
|태그|내용|
|---|---|
|`<dt></dt>`|정의 제목|
|`<dd></dd>`|정의 내용|
